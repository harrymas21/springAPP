# SpringCRUD
A Simple Spring MVC CRUD Application.

Database for the employees.

Don't forget to change the DB details
in the dispatcher-servlet.

```
      CREATE TABLE EMPLOYEE
  (
      ID INT(10) NOT NULL, 
      NAME VARCHAR(255), 
      DEPT VARCHAR(255), 
      AGE INT(10), 
      PRIMARY KEY(ID)
  ); 
  ```
